1. Clone the project
git clone <https://github.com/AjCodes/Smart-Roads>

2. Open the folder in VS Code

You will see:

arduino/
backend/
dashboard/

3. Go to your folder

Ethan: arduino/

AJ + Junior: backend/

Elias: dashboard/

4. Install dependencies (only if you're in backend or dashboard)

Backend:

cd backend
npm install


Dashboard:

cd dashboard
npm install

5. Start working

Only change files inside your own folder.

6. Save your work to GitHub
git add .
git commit -m "update"
git push

7. Always pull before starting work
git pull


8.You can always use GitHub desktop to make things easier 
https://desktop.github.com/download/
